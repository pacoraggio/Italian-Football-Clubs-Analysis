# Beyond the Golden Era: Tracking Italian Club Performance in Europe

## Overview

In recent years, many football pundits and commentators have been discussing the crisis of Italian football, from both a national team and club perspective. Some go further, declaring a broader crisis in the Italian football "movement" — encompassing youth development structures, educational frameworks, performance outcomes, and audience engagement across streaming, broadcast, and live attendance. A persistent myth also surrounds this debate: the tendency to compare modern Italian football unfavourably with that of the 1990s, often portrayed as the "golden era" of the Italian game.

In this project, we address a focused part of this broader discussion: the performance of Italian club teams in major European competitions. The analysis is primarily descriptive, defining key performance metrics and examining trends from the early 1980s — the pre-golden era — through to the most recent competitions. This initial analysis aims to surface first insights and identify notable trends that may warrant deeper investigation in future iterations.

## Project Structure

- `analysis.ipynb`: Main Jupyter notebook containing the data analysis and visualizations.
- `data/`: Directory containing UEFA European competition results data in CSV format, organized by time periods.
- `notebooks/`: Jupyter notebooks for data collection and preprocessing.
  - `data_scraping.ipynb`: Notebook for scraping UEFA data.
  - `data_wrangling.ipynb`: Notebook for data cleaning and preparation.
- `src/`: Source code directory.
  - Python scripts for data scraping and processing.
  - Additional data files.
